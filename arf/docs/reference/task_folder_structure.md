## Root File Restrictions

Only these files are allowed at the task root:

* `task.json`
* `task_description.md` (optional, spec v4 only)
* `step_tracker.json`
* `__init__.py`

Anything else at the task folder root is a verification error (`FD-E016`).

* * *

## Allowed Root Directories

* `plan/`
* `research/`
* `assets/`
* `results/`
* `corrections/`
* `intervention/`
* `logs/`
* `code/`
* `data/`

* * *

## Status-Dependent Requirements

| Status | Required Files |
| --- | --- |
| `not_started` | `task.json` + `task_description.md` (no directories or `step_tracker.json` yet) |
| `in_progress` | `task.json` + `step_tracker.json` + all directories; files accumulate as steps complete |
| `completed` | All mandatory files: `plan/plan.md`, `research/research_papers.md`, `research/research_internet.md`, all `results/*` files, at least one folder in `logs/steps/` |
| `cancelled` | `task.json` + `step_tracker.json` + files produced before termination |
| `permanently_failed` | `task.json` + `step_tracker.json` + files produced before termination |
